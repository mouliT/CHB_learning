## Rename convention

  `P[paper]_Fig[N][sub]_[short_description].png`

  Examples:
    P1_Fig3_CHB_topology.png
    P1_Fig5a_dc_link_balancing_waveform.png
    P2_Fig7_SRF_controller_block.png

## Steps

  1. Open each file - identify it against the paper (page number is in filename)
  2. Fill the Caption column below
  3. Rename the file to the convention above
  4. Add a \figentry{fig:Label}{filename_no_ext}{Caption}{width} to your .tex
